# 湖北大学流浪动物救助信息管理系统测试与部署指南

## 1. 系统环境要求

### 1.1 服务器环境
- **操作系统**：Windows/Linux/macOS
- **Node.js**：14.0+  
- **npm**：6.0+
- **MySQL**：5.7+或8.0+

### 1.2 客户端环境
- **浏览器**：Chrome、Firefox、Safari、Edge 最新版本
- **分辨率**：1024x768及以上

## 2. 系统部署

### 2.1 数据库部署
1. **安装MySQL**：
   - 下载并安装MySQL Community Server：https://dev.mysql.com/downloads/mysql/
   - 安装过程中设置root用户密码

2. **创建数据库**：
   - 登录MySQL：
     ```bash
     mysql -u root -p
     ```
   - 执行以下SQL语句：
     ```sql
     CREATE DATABASE IF NOT EXISTS hubu_animal_rescue DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
     ```

3. **导入数据库结构和数据**：
   - 执行以下命令：
     ```bash
     mysql -u root -p hubu_animal_rescue < database.sql
     ```

### 2.2 后端服务部署
1. **安装Node.js**：
   - 下载并安装Node.js：https://nodejs.org/en/download/
   - 验证安装：
     ```bash
     node -v
     npm -v
     ```

2. **安装依赖**：
   - 进入项目目录：
     ```bash
     cd hubu-animal-rescue
     ```
   - 安装依赖：
     ```bash
     npm install
     ```

3. **配置数据库连接**：
   - 修改 `config/db.js` 文件，设置正确的数据库连接参数：
     ```javascript
     const pool = mysql.createPool({
       host: 'localhost',
       user: 'root', // 数据库用户名
       password: 'root', // 数据库密码
       database: 'hubu_animal_rescue',
       waitForConnections: true,
       connectionLimit: 10,
       queueLimit: 0
     });
     ```

4. **启动服务**：
   - 执行以下命令：
     ```bash
     npm start
     ```
   - 服务将在 `http://localhost:3000` 启动

### 2.3 前端页面部署
- 前端页面为静态HTML文件，无需特殊部署
- 直接在浏览器中打开 `index.html` 即可访问系统

- **Apache JMeter**：用于模拟并发请求，测试系统性能

## 6. 故障排查

### 6.1 常见问题及解决方案

#### 6.1.1 数据库连接失败
- **症状**：后端服务启动失败，提示数据库连接错误
- **解决方案**：
  - 检查MySQL服务是否运行
  - 检查数据库连接参数是否正确
  - 检查数据库用户权限是否正确

#### 6.1.2 API接口返回错误
- **症状**：前端页面无法加载数据，或操作失败
- **解决方案**：
  - 检查后端服务是否运行
  - 检查数据库连接是否正常
  - 检查请求参数是否正确
  - 查看后端服务日志，定位错误原因

#### 6.1.3 前端页面加载缓慢
- **症状**：前端页面加载时间长，操作响应慢
- **解决方案**：
  - 优化前端代码，减少HTTP请求
  - 压缩静态资源
  - 优化数据库查询，提高API接口响应速度
